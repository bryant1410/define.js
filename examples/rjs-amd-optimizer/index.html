How to build DefineJS modules using r.js:

To build the application using r.js we first need to have access to API: 

npm install requirejs --save-dev

Then we need to define r.js build configuration file, which is much more convenient than passing manual arguments to r.js command:

build.js: 
({
  baseUrl: 'src',
  out: 'main.js',
  include: ['main'],
  wrap: true
})


build.sh: And now to take the final step we need to call the r.js command. As you see to make sure if requirejs is installed we have added one more line to it. 
#!/bin/bash

npm install --dev
./node_modules/requirejs/bin/r.js -o build.js

All you need to do is first adding execute permission to build.sh file and then execute it via terminal: 

./build.sh

Or simply run its content manually.
